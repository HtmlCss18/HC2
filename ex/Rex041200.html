HTML

기본 구조에 들어가는 태그

<!doctype> : 문서 유형을 지정하는 것. 웹 브라우저가 페이지를 어떻게 해석해야 하는지 알려준다.

ex) <!doctype html> : 이 페이지를 html로 해석해라.


<html> : 웹 페이지의 시작과 끝

웹 페이지는 <html>로 시작해서 </html>로 끝난다.


<head> : 웹 페이지의 정보, 문서에서 사용할 외부 파일들을 링크할 때 사용.

<title>, <meta>등이 들어간다.


<meta> : 문자 인코딩 및 문서 키워드, 요약 정보.

ex) <meta charset="utf-8"> : 웹 페이지의 문자 인코딩 방식을 utf-8로 지정해라.


<title> : 문서 제목.

<title>안의 내용이 웹브라우저의 제목 표시줄에 표시된다.

페이지를 방문하는 방문자나 검색엔진은 제목 표시줄의 제목을 보고 내용을 예측하기 때문에 잘 써야 한다.


<body> : 브라우저에 실제 표시되는 내용

input 태그

<input> : form의 요소중 하나.

            사용자가 정보를 입력하는 부분을 만들어야 할 때 사용한다.

            type에 따라 어떤 형식으로 정보를 받을지 결정된다.

            id 값을 통해 서로 구분할 수도 있다.

    ex) <input type="유형" name="속성값" id= "이름" value="초기값">


속성 값

type 속성 : 입력 요소의 유형.

  ex) <input type="유형">

value 속성 : 입력 요소의 초기값. 사용자가 변경 가능.

  ex) <input type="유형" value="초기값">

name 속성 : 입력 요소의 이름. 서버로 변수의 이름처럼 전달.

  ex) <input type="유형" name="속성값" id= "이름">

autocomplete 속성 : 자동 입력 완성.

autofocus 속성 : 페이지 로드 시 자동으로 입력 포커스.(페이지를 불러오자 마자 특정 부분에 마우스 커서가 표시되도록 하는 것)

placeholder 속성 : 입력 힌트를 희미하게 보여줌.(힌트 표시[필드 클릭 시 내용 사라짐])

readonly 속성 : 읽기 전용 필드.

  ex) <input type="text" value="only read" readonly>

required 속성 : 반드시 채워져 있어야 함.(필수 입력 창[빈 칸이면 안 넘어감])

  ex) <input type="text" name="name" required>

pattern 속성 : 허용하는 입력 형태를 정규식으로 지정.

  ex) <input type="text" pattern="(010)-\d{3,4}-\d{4}" title="형식 010-0000-0000">

max 속성 : <input> 필드의 최대값 지정.

min 속성 : <input> 필드의 최소값 지정.

maxLength 속성 : 텍스트 필드에 최대로 입력할 수 있는 문자의 개수 지정.

  ex) <input type = "text" maxlength = "20">

step 속성 : 숫자의 간격 설정


<input>이 date, datetime, datetime-local, month, week, time, number, range일 경우만 사용 가능.


type 유형

hidden : 서버로 보내는 값들을 보내는 필드.(사용자에게는 안 보임)

text : 한 줄짜리 텍스트를 입력할 수 있는 텍스트 상자.

search : 검색 상자.

tel : 전화번호 입력 필드.

url : url 주소 입력 필드.

email : 메일 주소 입력 필드.

password : 비밀번호 입력 필드.

number : 숫자를 조절할 수 있는 화살표.

range : 숫자를 조절할 수 있는 슬라이드 막대.

color : 색상표.

checkbox : 체크 박스.(2개 이상 선택 가능)

radio : 라디오 버튼.(1개만 선택 가능)

datetime : 국제 표준시(UTC)로 설정된 날짜와 시간.(연, 월, 일, 시, 분, 초, 분할 초)

datetime-local : 사용자가 있는 지역을 기준으로 한 날짜와 시간.(연, 월, 일, 시, 분, 초, 분할 초)

date : 사용자 지역을 기준으로 한 날짜.(연, 월, 일)

month : 사용자 지역을 기준으로 한 날짜.(연, 월)

week : 사용자 지역을 기준으로 한 날짜.(연, 주)

time : 사용자 지역을 기준으로 한 시간.(시, 분, 초, 분할 초)

button : 버튼.

file : 파일을 첨부할 수 있는 버튼.

submit : 서버 전송 버튼.

image submit : 버튼 대신 사용할 이미지.

reset : 리셋 버튼
