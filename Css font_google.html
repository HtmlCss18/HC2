구글에서 제공하는 폰트를 가져와 사용하는 법


1. 구글에 ‘구글 폰트’라 검색, 가장 위에 뜨는 ‘Browse Fonts’ 클릭합니다


2. 자신이 원하는 폰트를 선택한 뒤 오른쪽 상단의 버튼을 눌러 @inport를 누르고 코드를 확인합니다.


3. 코드를 보면 <style>태그에 적혀져 있는 코드는 style 안에 표시하면 됩니다.

<style>
@import url('https://fonts.googleapis.com/css2?family=Nanum+Pen+Script&display=swap');
</style>


4. CSS rules to specify families 아래 코드는 자신이 폰트를 지정하고 싶은 곳에 사용하면 됨.
(전체 폰트로 지정하고 싶다면 아래와 같이 사용하면 적용됨.)‘

body { font-family: 'Nanum Pen Script', cursive; }
