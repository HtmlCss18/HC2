border 속성은 테두리를 만드는 속성입니다.

부가 설명

border-width border-style border-color | initial | inherit

border-width : 선의 두께

border-style : 선의 모양

border-color : 선의 색상

initial : 기본값.

inherit : 부모 요소의 속성값을 상속받음.
